◆ _setISBN()

bool Livro::_setISBN ( std::string  isbn)
private

Configura o ISBN do livro, seguindo modelo de 13 dígitos "XXXXXXXXXXXX-X".

O código deve ser conferido a partir da fórmula a seguir, onde o último character deve bater com o calculado.

\[ x_{13}=10-((x_1+x_2+x_3+...+x_{11}+x_{12}) \text{ mod }10 ) \]

Parâmetros
isbnCódigo ISBN do livro
Retorna
True se bem sucedido, False se falhou
